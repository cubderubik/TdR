\part*{Abstract}

In this work I have learnt the techniques, methods and practices to solve the Rubik's cube blindfolded.
In addition, all the content of the practical part of the work has been carried out with the aim of improving the learning of these methods.
The practical part includes an application, a website and a tutorial where the whole process to solve the Rubik's cube blindfolded is explained.

\vspace{2cm}

En este trabajo ha sido realizado un aprendizaje de las técnicas, métodos y prácticas para resolver el cubo de Rubik a ciegas.
Además, todo el contenido de la parte práctica del trabajo has sido realizado con el objetivo de mejorar el aprendizaje de estos métodos.
La parte práctica incluye una aplicación, una web y un tutorial donde se explica todo el proceso para realizar el cubo de Rubik a ciegas.
